Jquery

Jquery è sicuramnte la libreria Javascripr più conosciuta ed utilizzata in ambito web. Uno dei molteplici vantaggi è quello di poter attraversare facilmente il DOM, per andare a selezionare a nostro piacimento gli elementi che ci interessano e renderli dinamici tramite una serie di metodi. Ci fornisce, inoltre, il metodo $.ajax() molto utilizzato per effettuare chiamate API a servizi esterni. Ad oggi lo utilizzeri certamente per la creazione di un sito web, consapevole però del fatto che esisteno altre librerie/framework altrettanto potenti.
